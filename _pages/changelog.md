# Changelog

A running history of releases for SMS Forwarder. The latest release is always at the top.

### `Initial Release`
# **Version 1.0**

SMS Forwarder is now live! Forward all your SMS, OTP and 2FA messages to your email, Telegram, WhatsApp, Slack or Discord channels. Easy setup in under five minutes — set it once and let it run automatically.

#### What's New
- Forwarding to **Email**, **Telegram**, **WhatsApp**, **Slack** and **Discord**
- **Forwarding rules** with sender and keyword filters
- **Schedule-based forwarding** so the automation only runs when you want it to
- **Built-in test message flow** to verify your setup end-to-end
- **In-app Shortcuts onboarding** that walks you through Apple Shortcuts setup
- **Privacy-first design** — no message content stored, RSA-OAEP-SHA256 wire encryption
- Full **light & dark mode** support
